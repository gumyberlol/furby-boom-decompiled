Decompilation of "Furby BOOM!" v1.9.0.

# Requirements
You will need [Unity 4.5.0](https://archive.org/details/old-unity-editor-archive) with a pro licence.

# Patches
- Removed ComAir
- Removed Volume and Headphone blocker

# Current bugs
- The furbling drink animation does not work.
- Some particles may be broken.

# Important stuff
You will need to grab the Furby BOOM! v1.9.0 .obb file and put the Generated folder into StreamingAssets. Without it, parts of the game will not work.
You can find the .obb in .xapk's, which you can easily find on sites like APKPure.
